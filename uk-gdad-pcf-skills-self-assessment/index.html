UK GDAD PCF skills self-assessment

Read about the United Kingdom (UK) Government Digital and Data (GDAD) Profession Capability Framework (PFC) and its skills and levels.

  • https://ddat-capability-framework.service.gov.uk/skills

This web page is a simple self-assessment, where you can rate your skill level from 0 to 4 for each of the UK GDAD PCF skills.

Skill levels:

  1. None
  2. Awareness
  3. Working
  4. Practitioner
  5. Expert

At the bottom of the web page, you can download your responses as a spreadsheet file, suitable for opening in any spreadsheet program that you like.

This web page is intentionally simple and focuses on providing a straightforward self-assessment experience. This web page doesn't track anything, doesn't access any data on your system, doesn't save anything to the cloud, etc.

Download your responses as a file.

  • The file is named "skills.tsv" and uses tab separated values.
  • It has a header row with the skill names, and a data row with your responses.
  • Suitable for opening in Apple Numbers, Google Sheets, Microsoft Excel, or similar.
  • Suitable for analysis with Python, Julia, R, or similar.
